## Typography
```scss
$ff-sans: Roboto, Helvetica, Arial, sans-serif;
$ff-fixed: 'Courier New', Courier, fixed;

$thin: 100;
$light: 300;
$regular: 400;
$bold: 700;
```

*Usage*

```scss
p.sans {
  font-family: $ff-sans;
  font-weight: $light;
}
```
